# Resources for testing

This folder should contain the files that are necessary for testing. All files should be grouped by types.

## Example

```
/resources
|-- images
|   |-- test-200x400.png
|   |-- nature-640x480.jpg
|   |-- 178x657.jpg
|-- videos
|   |-- sketch.mov
|   |-- sea.mp4
|-- docs
|   |-- cv.docx
|   |-- mail.pdf
```
